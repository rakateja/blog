About

Made Raka

Hi, I’m Made Raka, a software engineer with a passion for building reliable and maintainable software systems.

I write about topics I find interesting in software engineering — mostly around Go, distributed systems, software architecture, and design patterns. My goal is to share practical knowledge that can be applied in day-to-day engineering work.

What I write about

  • Go (Golang) — idiomatic patterns, standard library deep-dives, and real-world usage
  • Software Architecture — layered design, separation of concerns, dependency injection
  • Kubernetes & Docker — container orchestration and local development workflows
  • Software Engineering Practices — clean code, unit testing, and maintainability
